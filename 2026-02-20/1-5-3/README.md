# 📌 Simple Counter with Zustand & Emotion

React 기반 Counter 프로젝트입니다.  
Zustand를 활용하여 전역 상태(store)를 구성하고,  
Emotion(styled)을 사용하여 컴포넌트 단위 스타일링을 적용했습니다.

---

## ✨ Features

- 🔢 Zustand를 활용한 전역 상태 관리
- ➕ 숫자 증가 (increment)
- ➖ 숫자 감소 (decrement)
- 🔄 초기화 기능 (reset)
- 🧠 이전 상태 기반 상태 업데이트 로직 구현
- 🎨 Emotion(styled)을 활용한 컴포넌트 단위 스타일링
- 🎯 카드형 UI 구성
- ✨ 버튼 hover / active 인터랙션 적용
