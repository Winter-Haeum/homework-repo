# 📌 Post & Profile Card UI

React 기반의 컴포넌트 구조 설계 프로젝트입니다.  
ProfileCard와 Post 컴포넌트를 분리하여  
재사용 가능한 UI 구조를 설계하고  
props를 통한 데이터 전달 흐름을 구현했습니다.

---

## ✨ Features

- 👤 ProfileCard 컴포넌트 분리
- 📝 Post 컴포넌트 분리
- 📦 props를 통한 데이터 전달
- 🎨 CSS 스타일링 적용
- ♻️ 재사용 가능한 컴포넌트 구조 설계
