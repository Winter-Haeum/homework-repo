# 시간 배분하며 문제 풀기

문제 : 양꼬치

<br>

## 타이머 설정

- 읽기(2분)
- 계획(3분)
- 구현(10분)
- 검증(5분)

<br>

## 🔹 읽기 + 계획 : 10분 소요

### 1. 문제를 읽으며 파악함.

- 양꼬치 10인분마다 음료수 1개가 공짜
- 가격  
  - 양꼬치 1인분: 12,000원  
  - 음료수 1개: 2,000원

- 주어진 매개변수

  - `n` = 양꼬치 개수  
  - `k` = 음료수 개수  

- 출력값 : 총 지불 금액

<br>

### 2. 어떻게 문제를 풀어야 할까 고민함.

- 전체 가격을 구하면

```
n * 12,000 + k * 2,000 = 총 지불할 금액
```

- 그런데 공짜 음료수 값을 빼야 함.

  - 공짜 음료수 값을 구해보자.

  - n = 10 일 때 → 음료수 1개가 공짜

- 그러니까

```
(n / 10) * 2,000 = 공짜로 먹은 음료수 값
```

총 지불할 금액에서 공짜로 먹은 음료수 값을 빼면 됨.

<br>

## 🔹 작성해보자 : 7분 소요

1. 이건 반복 처리가 없으니까 for문은 아님.  
   그럼 const로 가자.

```js
const total = (n * 12000) + (k * 2000) - ((n / 10) * 2000);
```

2. 출력하는 건 함수니까 return을 해야지.

```js
return total;
```

3. 오류 발생…

## 🔹 문제 원인

- 소수점이 문제구나.

  - 소수점을 버려야 함.

  - 소수점 버리는 공식이 생각 안 나서 찾아봄.

<br>

## 🔹 수정 코드

```js
const total =
  (n * 12000) +
  (k * 2000) -
  (Math.floor(n / 10) * 2000);

return total;
```
