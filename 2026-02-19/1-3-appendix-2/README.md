# 📌 Mini Blog with React Router (1-3-appendix-2)

React 기반 Mini Blog 프로젝트입니다.  
React Router를 활용하여 홈, 글 목록, 글 상세 페이지를 구현하고,  
Emotion(styled)을 사용하여 컴포넌트 기반 스타일링을 적용했습니다.

---

## ✨ Features

- 🏠 홈 화면 구현
- 📋 글 목록 페이지 구현
- 🔎 동적 라우팅(`/posts/:id`)을 활용한 상세 페이지
- 🔁 useParams를 통한 URL 파라미터 읽기
- 🔙 목록으로 돌아가기 버튼 구현
- 🎨 Emotion(styled)을 활용한 컴포넌트 단위 스타일링
- 🎯 카드형 레이아웃 UI 구성
